Bagian 2: Benar atau Salah

21. DBCA dapat digunakan untuk menghapus database.
22. Database File Location, merupakan field yang dijumpai ketika melakukan instalasi Oracle yang berfungsi untuk menentukan lokasi penyimpanan database.
23. INSERT berfungsi untuk memasukkan record yang baru ke dalam sebuah tabel.
24. Aturan penamaan variabel dalam blok PL/SQL tidak boleh sama dengan nama kolom dalam tabel.
25. DBCA merupakan singkatan dari DATABASE CONFIGURATION ASSISTANT.
26. "Manipulate" termasuk dalam Reserved Word Oracle.
27. Aturan penamaan suatu variabel dalam blok PL/SQL harus sama dengan nama kolom pada tabel yang bersangkutan.
28. DATA WAREHOUSE termasuk dalam tipe database.
29. Group Function tidak dapat digunakan dalam blok PL/SQL.
30. Fungsi prefix dan sufix dalam penamaan sebuah variabel adalah untuk membedakan objek-objek database.
31. Database yang saling berelasi dan berintegrasi biasa disebut dengan Relational Database Management System(RDBMS).
32. Anonymous block adalah blok PL/SQL yang memiliki nama yang dapat mengambil parameter dan dapat dipanggil.
33. Data Warehouse berfungsi untuk menyimpan data dalam periode lama dan menerima data dalam operasi read.
34. Blok-blok yang terdapat dalam PL/SQL adalah Blok Utama dan Blok Sekunder.
35. Create and configure a database, adalah pilihan untuk melakukan instalasi software Oracle tanpa membuat penyimpanan databasenya.
36. Database Cloning adalah sebuah metode untuk membuat database dari awal.
37. Dalam struktur kontrol terdapat 3 bentuk percabangan(IF) yaitu IF, IF-THEN-ELSE dan NESTED IF.
38. Dengan memindahkan klausa END IF bersarang di akhir setiap kumpulan kondisi dan kegiatan yang lainnya, merupakan ciri khas yang dimiliki statement IF-THEN-ELSIF bila dibandingkan dengan NESTED IF.
39. Delimiter terdiri dari simbol tunggal dan simbol majemuk.
40. Fungsi REVOKE adalah untuk menarik hak akses dari user terkait.
